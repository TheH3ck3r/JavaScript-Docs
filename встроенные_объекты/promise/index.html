# –°–æ—Å—Ç–æ—è–Ω–∏—è Promise

–ü—Ä–æ–º–∏—Å –º–æ–∂–µ—Ç –Ω–∞—Ö–æ–¥–∏—Ç—å—Å—è –≤ –æ–¥–Ω–æ–º –∏–∑ —Ç—Ä—ë—Ö —Å–æ—Å—Ç–æ—è–Ω–∏–π:

  • fulfilled (–∏—Å–ø–æ–ª–Ω–µ–Ω) ‚Äî –æ–ø–µ—Ä–∞—Ü–∏—è –∑–∞–≤–µ—Ä—à–µ–Ω–∞ —É—Å–ø–µ—à–Ω–æ, –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è resolve()
  • rejected (–æ—Ç–∫–ª–æ–Ω—ë–Ω) ‚Äî –æ–ø–µ—Ä–∞—Ü–∏—è –∑–∞–≤–µ—Ä—à–∏–ª–∞—Å—å —Å –æ—à–∏–±–∫–æ–π, –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è reject()

1. –°–æ—Å—Ç–æ—è–Ω–∏–µ pending (–æ–∂–∏–¥–∞–Ω–∏–µ). –ü—Ä–∏ —Å–æ–∑–¥–∞–Ω–∏–∏ –Ω–æ–≤—ã–π Promise –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–∏ –æ–∂–∏–¥–∞–Ω–∏—è. –≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –æ–Ω –µ—â—ë –Ω–µ –∑–∞–≤–µ—Ä—à–∏–ª—Å—è.

–ü–æ–∫–∞ –Ω–µ –≤—ã–∑–≤–∞–Ω—ã resolve() –∏–ª–∏ reject(), –ø—Ä–æ–º–∏—Å –æ—Å—Ç–∞—ë—Ç—Å—è –≤ –æ–∂–∏–¥–∞–Ω–∏–∏ (pending).

2. –°–æ—Å—Ç–æ—è–Ω–∏–µ fulfilled (—É—Å–ø–µ—à–Ω–æ–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ). –ö–æ–≥–¥–∞ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è resolve(value), –ø—Ä–æ–º–∏—Å –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∏—Å–ø–æ–ª–Ω–µ–Ω (fulfilled), –∞ value –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è –≤ .then().

–ü–æ—Å–ª–µ resolve("üéâ –£—Å–ø–µ—Ö!") –ø—Ä–æ–º–∏—Å —Å—Ç–∞–Ω–æ–≤–∏—Ç—Å—è –∏—Å–ø–æ–ª–Ω–µ–Ω–Ω—ã–º.

3. –°–æ—Å—Ç–æ—è–Ω–∏–µ rejected (–æ—Ç–∫–ª–æ–Ω—ë–Ω). –ö–æ–≥–¥–∞ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è reject(error), –ø—Ä–æ–º–∏—Å –ø–µ—Ä–µ—Ö–æ–¥–∏—Ç –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –æ—Ç–∫–ª–æ–Ω—ë–Ω (rejected), –∞ error –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è –≤ .catch().
